# AI Research Assistant

A full-stack application with a React frontend for AI-powered research and document analysis, and a backend for file upload handling.

## Project Structure

```
└── src/                          # Frontend (React + TypeScript)
    ├── components/               # React components
    │   ├── ui/                   # Reusable UI components (shadcn)
    │   ├── QuestionInterface.tsx # Main research interface
    │   └── SystemArchitecture.tsx # System architecture display
    ├── pages/                    # Page components
    ├── hooks/                    # Custom React hooks
    ├── lib/                      # Utility functions
    └── assets/                   # Static assets
```

## Technology Stack

### Frontend
- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS** - Styling framework
- **shadcn/ui** - Component library
- **React Router** - Client-side routing
- **TanStack Query** - Data fetching and caching
